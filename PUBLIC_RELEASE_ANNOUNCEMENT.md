# AxiomMesh SDK 1.0.1

AxiomMesh SDK 1.0.1 is the first public baseline release of the EDP + MEP + SAVOIR stack.

It provides:

- a context-aware environment runtime
- a certainty-aware knowledge layer
- a JSON-RPC protocol for structured agent-environment interaction
- native multi-agent, multi-context, and federated execution support
- a drone/swarm reference implementation

This release stabilizes the protocol surface, exports the MEP schema and method catalog, adds chained provenance and recovery policies, and packages the stack with public release assets for publication.


MEP is presented here as an independent environment protocol, not an MCP overlay.
